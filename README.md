# QLA Frontend MVP (React + Vite)

Mobile-first frontend prototype for **QLA (Quality Life without Aging)**.

## Tech
- React + Vite
- React Router
- Local state + localStorage (dummy bookings)
- No backend, no database, no real authentication/API

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to Vercel
This project is Vercel-ready:
- Framework preset: **Vite**
- Build command: `npm run build`
- Output directory: `dist`
- SPA rewrite configured in `vercel.json`
